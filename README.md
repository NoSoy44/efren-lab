# efren-lab

Este es el código fuente del [Lab de mi portafolio](https://efrenmunoz.cl/lab), donde pruebo cosas que no caben en un proyecto de cliente: canvas, reconocimiento de voz, accesibilidad, IA aplicada, visión por computadora. Voy migrando cada demo a este repo por separado, a medida que tengo tiempo.

## Experimentos

| Demo | Estado | Stack |
|---|---|---|
| Modo alto contraste | ✅ live | CSS Custom Properties, React |
| Lector de página | 🔜 próximamente | Web Speech Synthesis API |
| Navegación por voz | 🔜 próximamente | Web Speech API |
| Game Over | 🔜 próximamente | Web Speech API, CSS Glitch, Web Audio |
| Ruido Generativo (Perlin) | 🔜 próximamente | Canvas API |
| Texto Destructivo | 🔜 próximamente | Canvas API, física de partículas |
| Partículas Interactivas | 🔜 próximamente | Canvas API, física 2D |
| Asistente IA | 🔜 próximamente | LLM (requiere API key propia) |
| Generador IA de diseño | 🔜 próximamente | LLM (requiere API key propia) |
| Control por gestos | 🔜 próximamente | MediaPipe Hands, WebRTC |

## Desarrollo

npm install
